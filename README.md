### Setup

#### 0 `npm` and `nodejs`

Make sure you have `npm` and `nodejs` installed. 

Otherwise run

```
sudo apt update
sudo apt install -y nodejs npm
```

#### 1 install dependencies

navigate to `{your_root_folder}/Peer2Paisa/backend`

and then run `npm install`
